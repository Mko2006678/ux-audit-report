# report.html a api/report.js – obsah pro upload na GitHub

Nahraj tyto soubory do repozitáře **ux-audit-report** (do kořene, vedle index.html).

---

## 1. Soubor: report.html (v kořeni repo)

V GitHubu: **Add file** → **Create new file** → název souboru: **report.html**

Zkopíruj celý obsah od řádku 1 do konce (od `<!DOCTYPE` po `</html>`).

---

## 2. Soubor: api/report.js

V GitHubu: **Add file** → **Create new file** → do názvu napiš: **api/report.js** (GitHub vytvoří složku api a do ní report.js).

Zkopíruj celý obsah od řádku 1 do konce.

---

**Nebo** nahraj z počítače: ve složce `AI UX audit/report-viewer/` máš soubor **report.html** a složku **api** s **report.js** – přetáhni je do GitHubu (Upload files).
